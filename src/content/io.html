Java Input/Output Programs

1. Enter a character using InputStreamReader

// Write a Java program to enter a character from user using InputStreamReader.
import java.io.*;

public class CharInput {
  public static void main(String[] args) throws IOException {
    InputStreamReader isr = new InputStreamReader(System.in);
    System.out.print("Enter a character: ");
    char ch = (char) isr.read();
    System.out.println("You entered: " + ch);
  }
}
Output: Displays the character entered by the user.

2. Enter a string using BufferedReader

// Write a Java program to enter a string from user using BufferedReader.

  }
}
Output: Displays the string entered by the user.

9. Get all characters from a file and display on screen

// Write a Java program to get all characters from the file and display on the screen.
import java.io.*;

public class FileCharDisplay {
  public static void main(String[] args) throws IOException {
    FileReader fr = new FileReader("sample.txt");
    int ch;
    while ((ch = fr.read()) != -1) {
      System.out.print((char) ch);
    }
    fr.close();
  }
}
Output: Displays all characters from sample.txt file.

10. Get all characters from file, count lines, words, characters

// Write a Java program to get all characters from the file,
// count number of lines, words, characters and display on the screen.
import java.io.*;

public class FileStats {
  public static void main(String[] args) throws IOException {
    BufferedReader br = new BufferedReader(new FileReader("sample.txt"));
    String line;
    int lines = 0, words = 0, chars = 0;

    while ((line = br.readLine()) != null) {
      lines++;
      words += line.split("\\s+").length;
      chars += line.length();
    }
    br.close();

    System.out.println("Lines: " + lines);
    System.out.println("Words: " + words);
    System.out.println("Characters: " + chars);
  }
}
Output: Displays line, word, and character count from sample.txt.
