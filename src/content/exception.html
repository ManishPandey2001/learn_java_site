1. Java Program that throws an exception and catch it using a try-catch block

public class TryCatchExample {
  public static void main(String[] args) {
    try {
      int result = 10 / 0;
      System.out.println("Result: " + result);
    } catch (ArithmeticException e) {

    }
  }

2. Java Program to create a method that throws an exception if the number is odd

public class OddNumberException {
  static void checkEven(int num) throws Exception {
    if (num % 2 != 0) {
      throw new Exception("Number is odd");
    }
    System.out.println("Number is even");
  }

  public static void main(String[] args) {
    try {
      checkEven(7);
    } catch (Exception e) {
      System.out.println("Exception: " + e.getMessage());
    }
  }
}
Output: Exception: Number is odd

4. Java Program to read numbers from file and throw exception if any number is positive

import java.util.*;

public class PositiveNumberException {
  public static void main(String[] args) throws Exception {
    Scanner sc = new Scanner(new File("numbers.txt"));
    while (sc.hasNextInt()) {
      int num = sc.nextInt();
      if (num > 0) {
        throw new Exception("Positive number found: " + num);
      }
    }
    System.out.println("All numbers are non-positive");
  }
}
Output: Throws exception if any number is positive

6. Java Program to read a list of integers from the user and throw an exception if any numbers are duplicates

import java.util.*;

public class DuplicateCheck {
  public static void main(String[] args) throws Exception {
    Scanner sc = new Scanner(System.in);
    Set set = new HashSet<>();
    System.out.println("Enter 5 integers:");

    for (int i = 0; i < 5; i++) {
      int num = sc.nextInt();
      if (!set.add(num)) {
        throw new Exception("Duplicate number found: " + num);
      }
    }
    System.out.println("All numbers are unique");
  }
}
Output: Throws exception if any number is repeated.

10. Java Program to throw SQL Query: insert, delete, update; if not successful then throw an exception

11. Java Program to show an example of throws Keyword

public class ThrowsExample {
  static void divide(int a, int b) throws ArithmeticException {
    if (b == 0) throw new ArithmeticException("Cannot divide by zero");
    System.out.println("Result: " + (a / b));
  }

  public static void main(String[] args) {
    try {
      divide(10, 0);
    } catch (ArithmeticException e) {
      System.out.println("Exception: " + e.getMessage());
    }
  }
}
Output: Exception: Cannot divide by zero

12. Java Program to show an example of multiple catch

public class MultipleCatch {
  public static void main(String[] args) {
    try {
      int[] arr = new int[3];
      arr[5] = 10 / 0;
    } catch (ArithmeticException e) {
      System.out.println("Arithmetic Exception: " + e.getMessage());
    } catch (ArrayIndexOutOfBoundsException e) {
      System.out.println("Array Index Exception: " + e.getMessage());
    } catch (Exception e) {
      System.out.println("General Exception: " + e.getMessage());
    }
  }
}
Output: Array Index Exception: Index 5 out of bounds for length 3

13. Java Program to show an example of nested try block

public class NestedTry {
  public static void main(String[] args) {
    try {
      try {
        int a = 10 / 0;
      } catch (ArithmeticException e) {
        System.out.println("Inner catch: " + e.getMessage());
      }

      int[] arr = new int[2];
      arr[5] = 100;
    } catch (ArrayIndexOutOfBoundsException e) {
      System.out.println("Outer catch: " + e.getMessage());
    }
  }
}
Output: Inner catch: / by zero
Outer catch: Index 5 out of bounds for length 2
